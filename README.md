# Portfolio Website - Shahzaib Hassan

A professional portfolio website showcasing my experience as a Full Stack Engineer with 7 years of experience.

## Features

- **Responsive Design**: Works seamlessly across all device sizes (desktop, tablet, mobile)
- **Accessibility**: Includes proper ARIA labels, alt tags, semantic HTML, and keyboard navigation
- **Interactive Elements**: Smooth animations, scroll effects, and dynamic counters
- **Modern Design**: Clean, professional UI with smooth transitions and hover effects

## Technologies Used

- **HTML5**: Semantic markup with accessibility features
- **CSS3**: Modern styling with CSS Grid, Flexbox, and custom properties
- **JavaScript**: Vanilla JS for interactivity and animations

## Key Sections

1. **Hero Section**: Introduction with call-to-action buttons
2. **About**: Professional summary with animated statistics
3. **Experience**: Timeline view of professional journey
4. **Skills**: Categorized technical skills
5. **Projects**: Featured open-source projects
6. **Contact**: Links to professional profiles

## Accessibility Features

- Semantic HTML5 elements
- ARIA labels and roles
- Alt text for all images and icons
- Keyboard navigation support
- Focus management for mobile menu
- Skip to main content link
- Reduced motion support for users with preferences

## Responsive Breakpoints

- Desktop: 1200px+
- Tablet: 768px - 1199px
- Mobile: Below 768px
